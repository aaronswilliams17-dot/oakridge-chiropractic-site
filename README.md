# Oakridge Chiropractic – Website Demo

Portfolio demo website for a chiropractic clinic, built to showcase responsive layout, clean UI, and user-focused UX decisions appropriate for a healthcare setting.

---

## Live Demo
https://aaronswilliams17-dot.github.io/oakridge-chiropractic-site/

---

## Overview
This project demonstrates a modern, calm, and professional website layout for a chiropractic practice.  
The focus is on clarity, accessibility, and mobile usability rather than heavy visuals or marketing language.

All business details are placeholders and used for demonstration purposes only.

---

## Features
- Responsive design for mobile and desktop
- Clear navigation and call-to-action buttons
- Hero section with image overlay and readable content
- “New Patient Visit” informational callout
- Services and pricing sections
- Appointment request form (demo only)
- Clean, clinical visual style

---

## UX / UI Focus
This project includes intentional UX/UI considerations such as:
- Visual hierarchy to guide users toward booking actions
- Calm, healthcare-appropriate color palette
- Semi-transparent hero card for readability over imagery
- Mobile-friendly spacing and tap targets
- Consistent layout patterns to reduce cognitive load

---

## Built With
- HTML5
- CSS3 (Flexbox, media queries)
- GitHub Pages

---

## Project Purpose
This site was created as a learning and portfolio project to practice:
- Real-world layout structure
- Responsive design techniques
- UX-focused visual decisions
- Clean, maintainable HTML and CSS
